# IT Support & Infrastructure Guide

### VPN Access
To access internal servers from a remote location, you must use the **GlobalProtect VPN**. 
* **Gateway:** `vpn.company.com` 
* **Credentials:** Use your standard SSO (Single Sign-On) credentials.
* **MFA:** Multi-factor authentication via the Duo app is required for every login.

### Hardware Requests
Standard laptop upgrades occur every **3 years**. 
* **Broken Hardware:** If your device is damaged, submit a ticket via the 'Jira Service Desk' immediately.
* **Peripherals:** Keyboards, mice, and monitors can be requested through the IT portal with manager approval.

### Software & Slack
* **Slack:** New employees are automatically added to the **#general** and **#announcements** channels.
* **Licenses:** For premium software (like JetBrains, Adobe, or Cursor Pro), submit a request via the IT portal. Manager approval is required for all paid licenses.
